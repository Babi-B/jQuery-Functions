The toggle() Funciton

The toggle() method toggles between hide() and show() for the selected elements.

This method checks the selected elements for visibility.
show() is run if an element is hidden.
hide() is run if an element is visible --- This creates a toggle effect. Check out the toggleClass() function

The toggleClass() Function

Toggles between adding and removing aclass name from an element

*******This DEMO alternates between two classes of different linear gradient direction

The fadeToggle() Function

The fadeToggle() method toggles between the fadeIn() and fadeOut() methods.

  • fadeToggle()
  • fadeToggle('slow')
  • fadeToggle(3000)

If the elements are faded out, fadeToggle() will fade them in.
If the elements are faded in, fadeToggle() will fade them out.

NOTE!!! Hidden elements will not be displayed at all (no longer affects the layout of the page).

The slideToggle() Function

The jQuery slideToggle() method toggles between the slideDown() and slideUp() methods

Click this to slide up/down instantly
slideToggle()
Click this to slide up/down slowly
slideToggle('slow')
Click this to slide up/down after 3secs
slideToggle(3000)

If the elements have been slid down, slideToggle() will slide them up.
If the elements have been slid up, slideToggle() will slide them down.
